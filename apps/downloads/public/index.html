Publishing New Versions

  1. Bump desktop app versions and push a desktop-v* git tag.
  2. GitHub Actions builds Windows + macOS installers and updater bundles.
  3. CI publishes files into /updates/pos/ and /updates/portal/ and updates latest.json.

This page hosts the live installer links and updater manifests consumed by both desktop apps.
