New Store Checklist

  1. Install POS Desktop on each cashier terminal and sign in with cloud credentials.
  2. (Optional) Install Admin Desktop for managers, or use the Manager button inside POS.
  3. Open cloud Admin to configure devices, users, and permissions.

Advanced (Updates)

  1. Merge updates to main (or push a desktop-v* git tag).
  2. GitHub Actions builds Windows + macOS installers and updater bundles automatically.
  3. CI publishes files into /updates/pos/, /updates/portal/ (Admin; alias /updates/admin/) and updates latest.json.

File listings and updater manifests: /updates/pos/, /updates/admin/.
