Publishing New Versions

  1. Build installers (Windows + macOS) and updater artifacts.
  2. Upload artifacts into /updates/pos/ and /updates/portal/.
  3. Update latest.json for each app.

For now, this page is just a safe place to host artifacts. The auto-updater wiring lives in each desktop app.
