# AH Trading Admin (Next.js + shadcn/ui)

This is the Admin ERP web app.

## Local Dev
1) Install dependencies:
   - `npm install`
2) Start dev server:
   - `npm run dev`

## Environment
- `NEXT_PUBLIC_API_BASE_URL` (default: `/api`)
- `API_PROXY_TARGET` (default: `http://127.0.0.1:8001` for local dev)
