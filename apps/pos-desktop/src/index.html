POS

Melqard POS Desktop

Connects to your POS server, queues sales offline

Setup

Recommended: Quick Setup (login once, pick companies, start). For on-prem you can still paste a Setup Pack.

Quick Setup (Recommended)

Enter your API URL and account, then select the companies you want the POS to issue invoices under. We will register this terminal as a POS device and fill the config automatically.

 
Setup Pack (on-prem / advanced)
Advanced (optional)

For each company you must paste a POS device token (created in Admin -> System -> POS Devices). After first Sync, the agent auto-pulls default warehouse + VAT settings.

Notes

  • This app starts 2 local agents: Official + Unofficial.
  • Unified POS opens at http://127.0.0.1:7070/unified.html.
  • Sales are queued locally if the server is unreachable; they sync when back online.
