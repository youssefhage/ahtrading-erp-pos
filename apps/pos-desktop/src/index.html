Melqard POS Desktop

Connects to cloud API, queues sales offline

Setup

Enter the API URL. For the cloud pilot use: https://app.melqard.com/backend

Advanced (optional)

For each company you must paste a POS device token (created in Portal -> System -> POS Devices). After first Sync, the agent auto-pulls default warehouse + VAT settings.

Notes

  • This app starts 2 local agents: Official + Unofficial.
  • Unified POS opens at http://127.0.0.1:7070/unified.html.
  • Sales are queued locally if the cloud is unreachable; they sync when back online.
