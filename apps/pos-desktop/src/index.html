AH Trading POS Desktop

Runs offline on LAN with Edge server

Setup

Enter the Edge API URL (LAN). Example: http://192.168.1.50:8001

Advanced (optional)

Device tokens and company IDs are configured in the agent config files. For pilot, we can set them from the Edge admin and paste them later if needed.

Notes

  • This app starts 2 local agents: Official + Unofficial.
  • Unified POS opens at http://127.0.0.1:7070/unified.html.
  • Sales are queued locally if Edge is unreachable; they sync when back online.
