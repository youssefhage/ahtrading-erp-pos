# Seeds

- `seed_coa_lebanon.sql` inserts the Lebanese COA template and accounts.
- `seed_account_roles.sql` inserts standard account role codes.
- `seed_companies.sql` inserts the two initial companies and main branches.
- `seed_company_coa.sql` creates company COA instances from the Lebanese template.
- Generate/update with: `python3 backend/scripts/generate_coa_seed.py`
