What To Install

Simple rule: Setup is for onboarding, POS is for cashiers, Admin is for managers. Download once. Apps auto-update in the background (when internet is available).

SU
Setup Desktop
Required
Install on: your laptop or the on-prem server machine

Starts the on-prem Edge server (Docker) and generates POS setup packs for terminals.

Windows (Setup Runner) Download for macOS
POS
POS Desktop
Required
Install on: every cashier terminal

Runs the local POS agents and opens the cashier UI. Works on LAN during internet outages.

Download for Windows Download for macOS
AD
Admin Desktop
Optional
Install on: manager computers (or a POS terminal if needed)

Dedicated Admin app wrapper. You can also open Admin from POS via the Manager button.

Download for Windows Download for macOS

New Store Checklist

  1. Install Setup Desktop on the on-prem/server machine (or your laptop) and run onboarding.
  2. Install POS Desktop on each POS terminal and paste the generated Setup Pack.
  3. (Optional) Install Admin Desktop for managers, or use the Manager button inside POS.

Advanced (Updates)

  1. Merge updates to main (or push a desktop-v* tag).
  2. GitHub Actions builds Windows + macOS installers and updater bundles automatically.
  3. CI publishes files into /updates/pos/, /updates/portal/ (Admin; alias /updates/admin/), /updates/setup/ and updates latest.json.

File listings and updater manifests: /updates/pos/, /updates/admin/, /updates/setup/.
