New store setup guide
  1. Install POS Desktop on each cashier terminal and configure credentials in POS Settings.
  2. No separate setup app is required.
  3. (Optional) Install Admin Desktop for managers, or use the Manager button inside POS.
  4. Open cloud Admin to configure devices, users, and permissions.
Developer & CI notes
  1. Merge updates to main (or push a desktop-v* tag).
  2. GitHub Actions builds Windows + macOS installers and updater bundles automatically.
  3. CI publishes files into /updates/pos/, /updates/portal/ (Admin; alias /updates/admin/) and updates latest.json.
